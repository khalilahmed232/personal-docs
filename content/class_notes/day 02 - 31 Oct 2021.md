## Modulus Arithemtic

( a * b ) % M = ( ( a % M ) * ( b % M) ) % M

( a + b ) % M = ( ( a % M ) + ( b % M) ) % M

( a - b ) % M = ( ( a % M ) - ( b % M) + M ) % M

( a / b ) % M = ( ( a % M ) * ( 1/ b % M) + M ) % M
= ( ( a % M ) * ( b^-1 % M)  ) % M
